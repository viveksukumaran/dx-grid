XO Real Work Assessment Details

Step 1: You will receive or might have received an email shortly about the work space setup created for your assessment.
Step 2: Follow the instructions given in the email. If you have not already done so, please:
  1. Download and install the Amazon WorkSpaces Client software. You'll use this software to connect to a remote PC on the internet that has been specially configured for this test.
  2. Watch this video demonstrating how to connect to the Amazon WorkSpace.
  3. Watch this video demonstrating how to use the Test Execution tool to conduct software tests.
  4. Watch this video demonstrating how to fail a test case and create a defect.
  5. Use the execution IDs 1376496768 & 1376496769 to practice the tests using XO TE Tool.
Please note that the XO TE tool login credentials are pre-configured. You should not enter or change the password.

During your testing assessment, your performance will be evaluated based on your ability to execute 5 test cases in less than 50 minutes.

Step 3: Please ensure that you complete the pre-work 72 hours prior to the assessment. Our experience shows that 100% of candidates who do not do the pre-work and practice with the test cases we provide fail the assessment.
Step 4: Finally, connect to the Work space before your scheduled assessment begins to make sure you are prepared to start when the session begins.
Best of luck!
